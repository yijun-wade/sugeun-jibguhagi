# 수근수근 집구하기

> 온라인 임장 도구 — 가고 싶은 집을 검색하면 가격 적정성, 실생활 여건, 실거주자 평가를 한 번에 보여줍니다

**부동산 매물 앱이 아닙니다. 거주 판단 도구입니다.**

---

## 제품 개요

아파트나 동네 이름을 검색하면 세 가지 질문에 답합니다.

1. **이 아파트, 지금 들어가도 되는 가격인가?** — 국토부 실거래가 기반
2. **거기 살면 실제로 어떤가?** — 교통 / 학군 / 편의 동네 정보
3. **다른 사람들은 이 동네를 어떻게 보나?** — 네이버 블로그/카페 실거주 후기

## 사용자 플로우

```
검색 (아파트명 or 동네명)
  ↓
집 평가 카드  ← 한줄 판단 + 가격 신호 + 생활 여건 + 실거주 한마디
  ↓
상세 임장 리포트
  ├ 가격 탭 — 월별 실거래 이력, 면적별 분포
  ├ 동네 탭 — 교통 / 학군 / 편의 / 개발 계획
  └ 이야기 탭 — 네이버 블로그/카페 실거주 후기
```

## 기술 스택

| 항목 | 기술 |
|------|------|
| 프론트엔드 | React 19, Vite |
| 배포 | Vercel (Serverless Functions) |
| 실거래가 | 국토교통부 실거래가 API (data.go.kr) |
| 실거주 후기 | 네이버 블로그/카페 검색 API |
| 동네 데이터 | 서울 70개+ 동 메타데이터 (자체 구축) |

## 로컬 실행

```bash
# 환경변수 설정
cp .env.example .env.local
# MOLIT_API_KEY, NAVER_CLIENT_ID, NAVER_CLIENT_SECRET 입력

# 설치 및 실행
npm install
npm run dev
```
